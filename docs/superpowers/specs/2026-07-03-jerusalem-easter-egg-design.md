# 예루살렘 이스터 에그 설계

## 목적

예루살렘에 전구 아이콘 마커를 고정 배치하고, 탭하면 ON-GI를 만든 배경 이야기를 보여주는 이스터 에그.

## 마커

- **위치**: 예루살렘 (lat: 31.7683, lng: 35.2137)
- **아이콘**: `/icon-bulb-sm.svg` (13x20px 전구)
- **크기**: 기존 기도 빛 마커의 core와 비슷한 사이즈
- **효과**: 은은한 골드 glow (항상 표시, 감쇠 없음)
- **동작**: 기도 데이터와 무관한 고정 마커, 항상 지구본에 표시
- **pointer-events**: auto (탭 가능)

## 모달

탭 시 전체 화면 모달로 표시. 스크롤 가능.

### 스타일
- 다크 배경 (`rgba(0, 0, 0, 0.95)`) + 골드 텍스트
- 상단에 전구 아이콘 표시
- 하단 닫기 버튼 또는 배경 탭으로 닫기
- `PrayerCompleteModal`과 유사한 톤

## 검증
1. `npm run build` 통과
2. 지구본에서 예루살렘 위치에 전구 마커 표시 확인
3. 마커 탭 → 모달 표시 → 스크롤 → 닫기 동작 확인
4. 다국어 전환 시 모달 내용 번역 확인
